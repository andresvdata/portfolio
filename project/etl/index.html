Extraction, Transformation and Load (ETL) project

Feb 1, 2021
Follow

Project to centralize information from various databases, through API queries and jobs built in the SQL STUDIO instance.
The database of this instance called Staging was the starting point, then through SSIS several data transformations were carried out to bring it to a new database called Data Universe.
Finally to create a star data cube it was necessary to create the dimensions and the facts, so this database can be connected to an SSAS.

Tools: SQL Management Studio,SQL Server Integration Services (SSIS)
Languajes: SQL
